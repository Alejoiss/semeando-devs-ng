O que é Semântica?

O HTML semântico é a base de uma web acessível e bem estruturada. Ao contrário de tags genéricas como div e span, tags semânticas descrevem o significado do conteúdo que elas envolvem.

code semantic-example.html 
<header>
  <nav>
    <ul>
      <li>Home</li>
    </ul>
  </nav>
</header>

<main>
  <article>
    <h1>O Poder da Semântica</h1>
    <p>Conteúdo importante aqui...</p>
  </article>
</main>

Utilizar a estrutura correta permite que leitores de tela guiem usuários com deficiência visual de forma eficiente, além de ajudar motores de busca como o Google a indexar melhor suas páginas, melhorando o SEO.
